# **FUNCTIONS IN SQL:**

- Types of Functions: 

  - System Defined Functions

    - Single Value Functions - Works row by row

      - String 
      - Number
      - Date
      - Null
      - Logical
      - Conversion
      - Rank

    - Multi Value Functions - Works on multiple rows at a time.

      - Aggregate Functions

        - Sum()
        - Max()
        - Min()
        - Count()
        - Avg()

  - User Defined Functions
